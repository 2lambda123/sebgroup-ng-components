Tooltip can be toggled using the show() and hide() functions. They can be useful in case you want to display the tooltip based on a condition and not the keyboard/mouse events.

In order to utilize the functions, you will need to use the tooltip component reference like in the below snippet.

{{ refTemplate }}

Then declare the tooltip ref

{{ refDeclaration }}

Then simply call show or hide

{{ "this.tooltipRef.show(); \nthis.tooltipRef.hide();" }}

Mouseout and Blur Events will still dismiss the tooltip.
